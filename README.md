# ft_transcendence

A full-stack multiplayer Pong platform built as the final project of the **42 Common Core**.
The project combines **frontend, backend, networking, real-time communication, and DevOps** to create a complete online gaming platform.

---

## 📌 Project Overview

**ft_transcendence** is a web application where users can register, connect with friends, and play **real-time Pong matches** directly in their browser.

The project brings together many core web development concepts including:

* 🔐 **Authentication & User Management** — registration, login, 2FA, and session handling
* 👫 **Social Features** — friends list, player profiles, online status
* 🎮 **Game Implementation** — real-time multiplayer Pong gameplay
* 🌍 **Single Page Application (SPA)** — smooth navigation without page reloads
* 🔌 **Real-time Communication** — WebSockets for live interactions
* 🐳 **Containerized Deployment** — Docker and Docker Compose
* 🏗 **Clean Architecture** — modular separation of frontend and backend

---

## 🎯 Goal

The objective of the project is not just to create a simple game, but to build a **complete online gaming platform** where users can:

* Create accounts and manage profiles
* Add friends and see their online status
* Play real-time matches
* Track match history and statistics
* Participate in tournaments

The application is built as a **Single Page Application (SPA)** using **Vanilla TypeScript** and **TailwindCSS**.

---

## ✨ Features

* User authentication (login & registration)
* Friend system with live status
* Real-time Pong gameplay
* Match history and statistics
* Tournament mode
* Custom game maps
* Interactive player dashboard
* Responsive user interface
* SPA client-side routing

### Frontend

* **Vanilla TypeScript**

* REST API communication

* **Docker**

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/GuillemFo/ft_transcendence.git
cd ft_transcendence
```

Build and start the project:

```bash
make
```

Then open the application in your browser:

```
http://localhost:3000
```
